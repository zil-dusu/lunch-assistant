## 3. 操作指南

### 3.1 随机推荐一道菜

1. 选择分类（可选）：点米饭/面食/小吃中任一个
2. 选择辣度（可选）：点辣/不辣中任一个
3. 点 **"🔀 随机推荐"** 按钮
4. 等待滚动动画结束，查看推荐结果

### 3.2 从收藏中推荐

1. 点 **"⭐ 从我的最爱推荐"** 按钮
2. 从你已经收藏的菜品中随机选一道
3. 如果还没收藏过，会提示"还没有收藏菜品"

### 3.3 换一个

1. 推荐结果出来后，点 **"🔄 换一个"**
2. 重新随机推荐，**短时间内不会重复**同一道菜

### 3.4 收藏菜品

1. 推荐结果出来后，点 **"⭐ 加入最爱"**
2. 收藏成功，到"最爱"页面可以看到
3. 再次点击可 **取消收藏**

### 3.5 查看热度榜

1. 点底部 **"🔥热度榜"** 标签
2. 查看被推荐最多的前 5 道菜

## 4. 常见问题

| 问题 | 解答 |
|------|------|
| 为什么换了个浏览器收藏没了？ | 收藏数据存在**浏览器本地**，换浏览器相当于换了个用户 |
| 清除浏览器数据后收藏还在吗？ | **不在**。UUID 和收藏关联都存储在 localStorage 中 |
| 为什么换一个总是同一道菜？ | 可能只有 1 道菜符合筛选条件，调整筛选条件试试 |
| 页面打不开？ | 检查网络连接，或联系管理员确认服务器状态 |

## 5. 快捷方式

在手机浏览器中打开后，可将该页面**添加到手机桌面**，像 App 一样使用：

- **iPhone**：Safari → 底部分享按钮 → "添加到主屏幕"
- **Android**：Chrome → 右上角菜单 → "添加到主屏幕"
